Paragraph Tag
The <p> tag is used to define paragraphs in HTML. It represents a block of text that is separated from adjacent text.

Aliquam nec scelerisque tortor. Praesent in aliquam sem. Curabitur diam magna, suscipit eu odio feugiat, malesuada elementum nisl.

Bold (b) Tag
The <b> tag is used to make text bold. It's often used for emphasizing text without implying any additional importance.
bold
Italic (i) Tag
The <i> tag is used to apply italic styling to text. It's often used for providing emphasis or indicating a different tone.
Italic
Underline (u) Tag
The <u> tag is used to underline text. It's often used for indicating hyperlinks or emphasizing text.

underline

Strikethrough (s) Tag
The <s> tag is used to create strikethrough text, indicating that the text is no longer relevant or has been deleted.

strikethrough

Emphasis (em) Tag
The <em> tag is used to emphasize text, typically displayed in italic style. It indicates a level of stress or importance.

emphasize

Small Tag
The <small> tag is used to make text appear smaller than the surrounding text. It's often used for fine print or disclaimers.

smaller

Strong Tag
The <strong> tag is used to give strong importance to text, and it is typically displayed in bold.

strong

Superscript (sup) Tag
The <sup> tag is used to render text in a superscript style, often used for mathematical notations or footnotes.

superscript

Subscript (sub) Tag
The <sub> tag is used to render text in a subscript style, often used for mathematical notations.

subscript

Deleted (del) Tag
The <del> tag is used to represent deleted or removed text. It's often used to indicate that the text has been struck through.
Delete
Inserted (ins) Tag
The <ins> tag is used to represent inserted or added text. It's often used to indicate that the text has been underlined.

inserted

Anchor Tag (Link)
The <a> tag, also known as the anchor tag, is used to create hyperlinks. It defines a clickable link to another web page or resource.

link

Abbreviation Tag
The <abbr> tag is used to define abbreviations or acronyms. It can provide additional information when the user hovers over the abbreviation.

Abbreviation

Mark Tag
The <mark> tag is used to highlight parts of text. It's often used to indicate search results or relevant portions of content.

highlight

Detail tag
The <details> tag is used to create a disclosure widget from which the user can obtain additional information or controls.
Epcot Center

Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.

Pre Tag
The <pre> tag is used to define preformatted text. Text within this tag is displayed in a fixed-width font, and it preserves both spaces and line breaks. 
Text in a pre element is displayed in a fixed-width font, and it preserves both      spaces and line breaks
Kbd Tag
The <kbd> tag is used to define keyboard input. It typically represents user input as keys, buttons, or other interactive controls.

Press Ctrl + C to copy text (Windows).

Press Cmd + C to copy text (Mac OS).

Var Tag
The <var> tag is used to define variables in mathematical or programming contexts.

The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.

Samp Tag
The <samp> tag is used to define sample output from a computer program or script.

File not found.
Press F1 to continue

Address Tag
The <address> tag is used to define contact information for the author or owner of a document.
1 Infinite Loop
Cupertino, CA 95014
United States
Cite Tag
The <cite> tag is used to reference the title of a creative work, such as a book or movie. It typically appears in conjunction with a quotation.

"Code is poetry." --Automattic

Code Tag
The <code> tag is used to define inline code snippets within a text paragraph.
